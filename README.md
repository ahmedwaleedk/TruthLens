# 🕵️ TruthLens

## Multi-Modal Fake News Detection System

TruthLens is a multi-modal fake news detection system developed to help users identify whether news content is **Real** or **Fake** by analyzing both textual and visual information.

With the rapid spread of misinformation across social media and online news platforms, TruthLens aims to provide an intelligent solution for detecting misleading content quickly and accurately.

---

## 🚀 Features

- Detects fake and real news articles.
- Analyzes textual news content using NLP techniques.
- Supports image analysis for visual misinformation detection.
- User-friendly interface.
- Provides prediction results in real time.

## 🎯 How It Works

1. User submits news content.
2. The system processes text and images.
3. Machine learning models analyze the content.
4. The system classifies the news as **Real** or **Fake**.
